# angular2-tour-of-heroes

```bash
# Install [CLI tool for Angular2](https://github.com/angular/angular-cli)  
npm install -g angular-cli

# crear new angular2 project  
ng new PROJECT_NAME
cd PROJECT_NAME
ng serve
# Navigate to http://localhost:4200/
```
